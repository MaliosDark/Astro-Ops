Bonk Raiders WebSocket Server

Checking server status...

This server provides real-time communication for the Bonk Raiders game, enabling:

  • Player status updates
  • Raid notifications
  • In-game chat
  • Real-time battles
  • Leaderboard updates

To connect to the WebSocket server from your client:

const ws = new WebSocket('wss://webrtc.bonkraiders.com/ws');

// Authenticate after connection
ws.onopen = () => {
  ws.send(JSON.stringify({
    type: 'auth',
    data: {
      token: 'your-jwt-token'
    }
  }));
};

// Handle messages
ws.onmessage = (event) => {
  const message = JSON.parse(event.data);
  console.log('Received:', message);
};

For more information, please refer to the documentation.
